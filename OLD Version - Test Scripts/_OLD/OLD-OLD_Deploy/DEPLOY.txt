:CONFIGURATION:
 
Faire les mises à jours (Windows + Dell)
 
Nommer le poste (voir Identifiant du domaine) => System Properties Advanced

Créer et coller l’étiquette de nommage sur le poste
 
Mettre le poste sur le domaine : ceva.net 

Mettre le poste dans la bonne OU sur l'AD (de base le poste arrive dans l'OU Computers)

Faire un reboot
 
Forcer l’application des GPO suite au changement d’unité d’organisation => gpupdate 

Penser à bien mettre le mot de passe au compte CEPRT
